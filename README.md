# forum-front-end-vue
+ 功能介紹：
+ 練習 vue2 / vuex 的使用
+ 練習串接後端API
+ 優化使用者體驗，例如：
   1. 尚未取得資料前，顯示讀取中的頁面
   2. 防止使用者連續點擊發送請求的按鈕（加入最愛或 like）
   3. 如有空的餐廳類別，顯示相關說明
